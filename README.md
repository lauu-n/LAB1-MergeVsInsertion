# LABORATORIO I: MERGE vs INSERION SORT

En este laboratorio se realizó un análisis comparativo entre los algoritmos de ordenamiento Insertion Sort y Merge Sort, con el objetivo de evaluar su desempeño en términos de tiempo de ejecución al trabajar con diferentes tamaños de datos, utilizando Python.

Durante el experimento se midió el tiempo que tardaba cada algoritmo en ordenar los datos, dentro del programa. Los resultados obtenidos fueron registrados en tablas y representados mediante gráficas con el fin de visualizar el comportamiento de cada algoritmo a medida que aumenta el tamaño del arreglo.

A partir del análisis de los resultados se pudo observar que Merge Sort presenta un crecimiento más controlado en el tiempo de ejecución, mientras que Insertion Sort aumenta considerablemente su tiempo cuando el tamaño del arreglo crece.
